## Getting Started with Configuration
To begin configuring the bot, you will need to run the ``/settings config`` command.

### Requirements  
- You must have the **Manage Server** permission to access configuration settings.  
- Ensure the bot has the necessary permissions and has a high role.

### Running the Setup  
* Use ``/settings config``  

    - Click **continue** to navigate through the setup process.  
    - Follow the on-screen instructions to adjust settings according to your needs.  
    - Once completed, all changes will be saved automatically.  

You have now accessed the Configuration panel and can begin customizing the bot!
